# DM Skills

Public install repository for the DM Skills framework.

These skills are built for AI coding assistants and AI harnesses that support
the skills standard. They cover the full direct mail workflow:

- `dm-plan`
- `dm-create`
- `dm-execute`
- `dm-measure`
- `dm-strategist`

## Install

```bash
npx skills add dm-skills/skills
```
